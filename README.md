# Contact Directory ( .NET MAUI )

A simple mobile contact directory application built with .NET MAUI. The app displays a list of sample contacts grouped alphabetically and allows users to view detailed contact information using stack-based navigation.

## Features

- Displays 15 contacts with images and names  
- Contacts grouped alphabetically using CollectionView  
- Tap a contact to view full details:
  - Name  
  - Email  
  - Phone number  
  - Description  
- Stack-based navigation between pages  
- Back navigation from the contact details page  

## Built With

- .NET MAUI  
- C#  
- XAML
